How it works

  1. 01
    Register
    Create an account and choose where you want alerts: Telegram, Discord, or Email.
  2. 02
    Add URLs
    Paste any URL into your dashboard. PingWatch starts checking it every 5 minutes immediately.
  3. 03
    Get alerted
    When a site goes down you get a message. When it recovers — another message with downtime duration.

Zero friction. Zero infrastructure.

5-minute pings

Netlify Scheduled Function on */5 * * * *. Change the cron to any interval you need.

Down + Recovery alerts

You get a message when the site falls AND when it comes back, with exact downtime in minutes.

Incident log

Every outage is stored in Supabase with started_at, resolved_at, and duration_seconds.

No build step

Pure HTML + JS on the frontend. Supabase via CDN. Deploy by drag-and-drop to Netlify.
